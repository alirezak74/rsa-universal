# RSA Faucet

A testnet faucet service for the RSA CRYPTO token (code: RSA) on the RSA Chain. Request test RSA CRYPTO tokens for development and testing.

## Features
- Dispense testnet RSA tokens
- Rate limiting to prevent abuse

## Getting Started

```sh
npm install
node index.js
```

## Configuration
- Set environment variables as needed (see `.env.example` in the project root).
